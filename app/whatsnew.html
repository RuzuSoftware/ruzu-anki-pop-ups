Ruzu Anki pop-ups

What's New?


Version 2.0.1
  • Fixed compatibility issue with AnkiConnect.

Version 2.0.0
  • All text based Anki decks and models are now supported.
  • Added advanced options that allow configuration of card display for each model.

Version 1.1.0
  • Unescape special characters that look bad in pop-ups.
  • Remove dependency on bootswatch for styles.

Version 1.0.0

Initial version that includes:

  • Test notifications
  • Options to select the deck and test frequency
  • Can enable/disable notifications quickly and easliy via options menu
  • See the status of the app at a glance by looking at the icon
  • Keyboard shortcut to enable / disable pop-ups (Default: Ctrl+Shift+1)
  • Keyboard shortcut to show the next card early (Default: Ctrl+Shift+2)
  • Support for Anki decks using the 'Basic' model, or other models that use 'Front' and 'Back' fields.

If you like this extension, please consider giving us some support. The support page can also be accessed by right-clicking the icon.
