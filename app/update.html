Update AnkiConnect


This page can be used to easily update AnkiConnect. Follow the steps below to perform an update.

  1. Open Anki
  2. Click the button below to prompt AnkiConnect to update
  3. Navigate to Anki and confirm that you would like to update AnkiConnect
  4. Restart Anki

For more info on how to use Ruzu Anki pop-ups, please look at the getting started page.
