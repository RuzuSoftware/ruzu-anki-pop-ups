Ruzu Anki pop-ups

Getting Started

  1. Ensure that you have downloaded version 6+ of the AnkiConnect Anki plugin. Download it here
  2. Start Anki on you desktop
  3. Click the icon to open the options (Options can also be accessed from chrome://extensions/)
  4. Select the deck which you would like to study
  5. Select the test frequency
  6. Enable test notifications by ticking the box that looks like this:
  7. Click 'Save' to save your changes

Please note that this extension currently supports text based Anki decks and models only.

Other Features

  1. You can change the deck or test frequency at any time using the dropdowns in the options menu
  2. Press the 'Show next question' button to force the next question pop-up to be displayed
  3. You can enable / disable pop-ups using the keyboard shortcut (Default: Ctrl+Shift+1)
  4. You can show the next question early using the keyboard shortcut (Default: Ctrl+Shift+2)
  5. Visit chrome://extensions/ and click 'Keyboard shortcuts' (at the bottom of the page) to configure shortcuts
  6. Use the advanced options page to configure how card fields are displayed for each model.

If you like this extension, please consider giving us some support. The support page can also be accessed by right-clicking the icon.
